#  Property Value Estimator

A professional Machine Learning web application built using Python, Streamlit, and scikit-learn that predicts residential property prices using a Linear Regression model.

---

##  Features

- AI-powered Property Price Prediction
- Interactive Streamlit Dashboard
- Real-time Predictions
- Linear Regression Model
- Dataset Preview & Analytics
- Model Evaluation Metrics (R², MAE, RMSE)
- Interactive Charts & Visualizations
- Download Dataset Feature
- Professional Dark-Themed UI

---

##  Machine Learning Model

This project uses a *Linear Regression* model to predict property prices based on:

- Square Feet
- Bedrooms
- Bathrooms

---

##  Model Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

---

##  Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- scikit-learn
- Matplotlib
- Seaborn

## ▶️ How to Run the Project

### Install Dependencies

bash
pip install -r requirements.txt


### Run the App

bash
streamlit run app.py
